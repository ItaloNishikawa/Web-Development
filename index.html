Presentation

Desenvolvimento Web é uma disciplina fundamental para aqueles que desejam aprender a criar websites e aplicações web modernas. O objetivo dessa disciplina é ensinar os alunos a desenvolverem sites dinâmicos e interativos, capazes de oferecer uma experiência de usuário única e agradável.

Durante o curso, os alunos aprenderão conceitos fundamentais de HTML, CSS e JavaScript, as linguagens básicas do desenvolvimento web, e também serão apresentados a outras tecnologias avançadas, como frameworks de front-end e back-end.

Além disso, os alunos também aprenderão a criar sites responsivos, que se adaptam a diferentes tamanhos de tela e dispositivos, como celulares e tablets. Isso é essencial no mundo atual, onde a maioria dos usuários acessa a internet por meio de seus dispositivos móveis.

Ao final da disciplina, espera-se que os alunos tenham desenvolvido habilidades práticas e teóricas necessárias para criar sites modernos, elegantes e funcionais, capazes de atender às necessidades dos usuários e das empresas em um mundo cada vez mais conectado e digital.

Conteúdo do programa

  • Introdução ao desenvolvimento web
  • HTML e estruturação de conteúdo
  • CSS e estilização de páginas web
  • JavaScript e programação front-end
  • Frameworks front-end (ex: React, Angular, Vue)
  • Introdução ao desenvolvimento back-end
  • Linguagens back-end (ex: PHP, Ruby, Python)
  • Frameworks back-end (ex: Laravel, Ruby on Rails, Django)
  • Banco de dados e SQL
  • Integração front-end e back-end
  • Desenvolvimento de sites responsivos
  • Testes e depuração de sites e aplicações web
  • Segurança em desenvolvimento web
  • Boas práticas de desenvolvimento web
